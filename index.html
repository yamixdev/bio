Zapret

v1.9.0b

Что нового

Исправлена проблема, из-за которой не работал профиль обхода UDP GameFilter (1024-65535). Теперь сервера в Battlefield и других играх должны работать стабильно.

Важно: Перед началом включите Secure DNS в настройках браузера (или Windows 11).
1

Скачивание и Распаковка

Скачайте архив. Нажмите ПКМ по архиву -> Свойства -> Галочка "Разблокировать" (обязательно!). Распакуйте папку в корень диска (без пробелов).

2

Поиск рабочей стратегии

Запускайте файлы general-*.bat по очереди от имени администратора. Не закрывайте черное окно и проверяйте YouTube/Discord.

⚠️ Пробуйте ВСЕ файлы (general-alt, general-fake, general-mgm и т.д.) пока не заработает!

3

Автозапуск (Service)

Нашли рабочий вариант? Закройте черное окно. Запустите service.bat -> выберите Install Service -> выберите ту стратегию, которая сработала.

Не грузит профиль? Мобильные операторы могут блокировать загрузку. Подключитесь к Wi-Fi, загрузите профиль, и потом пользуйтесь на мобильном интернете.

Инструкция:

  1. Нажмите на любой блок выше (ссылка скопируется).
  2. В приложении нажмите "+" -> "Добавить из буфера".
  3. Нажмите большую кнопку подключения посередине.
  4. Выбор сервера: Внизу нажмите Прокси. Сверху нажмите (Молния) — это проверит пинг. Выбирайте сервер с зеленым значением.
