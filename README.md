Персональный хаб и портфолио.

Разработано с использованием **Native Web Technologies** (Vanilla Stack).
Сайт адаптирован для всех устройств и поддерживает только **Dark** тему!

### Технический стек
*   **Core:** HTML5, CSS3 (Variables, Flex/Grid), ES6+ JavaScript.
*   **Architecture:** Zero-dependency, компонентный подход.
*   **Features:**
    *   Асинхронная работа с API (Lanyard, Cataas).
    *   Кастомные CSS-анимации (60fps).
    *   Полная адаптивность (Mobile First).
