# ilushadev.github.io

Привет! Это персональная страница Ильи — минималистичное портфолио, собранное вручную на чистом HTML/CSS/JS.

- Стек: HTML5 · CSS3 · JavaScript (Vanilla)
- Шрифты: JetBrains Mono + Inter
- Тема: тёмная, с аккуратными анимациями и модальными окнами

Зачем это сделано?
- Потому что красиво выглядит и приятно кодить. Здесь нет фреймворков — всё на чистом коде.

Немного юмора:
> Этот сайт — как швейцарский нож в кармане программиста: маленький, стильный и всегда под рукой. Если он сломается — значит ты делал апдейты слишком быстро.

Как запустить локально:
```powershell
# запустить простой HTTP сервер (Windows PowerShell)
python -m http.server 8000;
# затем открыть http://localhost:8000
```
